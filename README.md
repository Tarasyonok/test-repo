# техстек

### **1. Регистрация и авторизация**

* **Язык/Фреймворк**:**Go (Golang)**&#x441; использованием**Gin**или**Echo**.
* **Причина**: Go отлично подходит для реализации высоконагруженных систем, таких как аутентификация и авторизация. Он обеспечивает высокую производительность и низкую задержку. Gin или Echo — легкие и быстрые фреймворки для создания RESTful API.
* **Микросервисы**:
  * Регистрация и авторизация через email/логин и пароль.
  * Интеграция с социальными сетями (Google, GitHub, LinkedIn).
  * Двухфакторная аутентификация (2FA).
  * Восстановление пароля.

***

### **2. Профиль пользователя**

* **Язык/Фреймворк**:**Java (Spring Boot)**.
* **Причина**: Spring Boot предоставляет мощные инструменты для работы с базами данных, валидацией данных и сложной бизнес-логикой. Это важно для управления расширенной информацией о пользователе.
* **Микросервисы**:
  * Управление профилем пользователя (личные данные, фотографии, местоположение, описание, навыки, опыт работы, образование, ссылки).
  * Настройка приватности профиля.
  * Управление портфолио проектов.

***

### **3. Лента новостей**

* **Язык/Фреймворк**:**Python (FastAPI)**.
* **Причина**: FastAPI обеспечивает высокую производительность и простоту разработки API. Python хорошо подходит для работы с персонализированными данными и алгоритмами рекомендаций.
* **Микросервисы**:
  * Генерация персонализированной ленты новостей.
  * Публикация текстовых сообщений, изображений, ссылок.
  * Лайки, комментарии, репосты.
  * Уведомления о новых событиях.
  * Фильтрация новостей по категориям.

***

### **4. Поиск**

* **Язык/Фреймворк**:**Go (Golang)**&#x441; использованием**Elasticsearch**.
* **Причина**: Go обеспечивает высокую производительность для поисковых операций, а Elasticsearch — это мощный инструмент для полнотекстового поиска и фильтрации.
* **Микросервисы**:
  * Поиск пользователей, проектов, вакансий.
  * Расширенный поиск с фильтрами и сортировкой.

***

### **11. API Gateway**

* **Язык/Фреймворк**:**Go (Golang)**&#x441; использованием**gRPC**или**REST**.
* **Причина**: Go обеспечивает высокую производительность и низкую задержку, что критично для API Gateway. Он также хорошо интегрируется с другими микросервисами.

***

### **12. Фронтенд**

* **Язык/Фреймворк**:**TypeScript + Vue 3**.
* **Причина**: Vue 3 — это современный и легкий фреймворк для создания пользовательских интерфейсов. TypeScript обеспечивает строгую типизацию, что упрощает разработку и поддержку кода.

***

### **Итоговое распределение:**

* **Go (Golang)**: API Gateway, регистрация и авторизация, поиск, рейтинги и достижения.
* **Java (Spring Boot)**: Профиль пользователя, сообщения, управление проектами, функции бизнес-инкубатора, вакансии.
* **Python (FastAPI)**: Лента новостей, события, деловые знакомства, блоги.
* **TypeScript + Vue 3**: Фронтенд.
